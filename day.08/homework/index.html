BR Tag Explanation

Explanation of BR Tag

The
tag is used to insert line breaks.
It helps in breaking the text into multiple lines.

Example with
tag:
This is line 1.
This is line 2.
This is line 3.

Example without
tag:

This is line 1. This is line 2. This is line 3.
